# Client

## Overview
The Client Management module in our ERP system allows users to efficiently manage client information, track relevant details, and maintain organized records. This module is crucial for handling client data, including contact information, payment schedules, and document management.

### How to Create New Client

**Steps to Create a New Client**

Click on New Client: To create a new client, navigate to the Client section and click on the 'New Client' button.

### Client Details

<div style="text-align:left;">

<strong>Client Id:</strong>
This is a system-generated sequence number for the client.

   >	The sequence can be defined in **Administration** **->** **Basic** **->** **Document Sequence**.

<strong>Client Name:</strong>
Enter the name of the client.

<strong>Country:</strong>
Select the country of the client.

   >	Countries can be added from **HR** **->** **Basic** **->** **Country**.

<strong>Emirates:</strong>
Select the emirates of the client.

   >	Emirates can be added from **HR** **->** **Basic** **->** **Emirates**.

<strong>Customer:</strong>
Used to connect the client with a CRM customer.

   >	Customers can be added from **CRM** **->** **Basic** **->** **Customer**.

<strong>Currency:</strong>
elect the currency of the client. The default currency under the client's country will be shown.<

<strong>Payment Schedule:</strong>
Specify the payment schedule of the client, indicating the expected payment date range (e.g., 30 days, 90 days, etc.).

   >	Payment schedules can be added from **HR** **->** **Basic** **->** **Payment Schedule**.

<strong>Grades:</strong>
Define the quality of the client based on grades.

   >	Grades can be added from **HR** **->** **Basic** **->** **Grades**.

<strong>TRN (Tax Registration Number):</strong>
TRN stands for Tax Registration Number under laws.

<strong>Website:</strong>
Enter the website details of the client.

<strong>Account:</strong>
Select the tax account of the client. You can use the 'Generate Account' button to create an account for the client and assign it.</li>

   >	Accounts can also be created from **Accounts** **->** **Basic** **->** **Chart of Accounts**.

### Contact Details

<strong>Telephone:</strong>
Enter the telephone number of the client.

<strong>Mobile No:</strong>
Enter the mobile number of the client.

<strong>Location:</strong>
Enter the location of the client.

   >	Locations can be created from **HR** **->** **Basic** **->** **Locations**.

<strong>Address:</strong>
Enter the address of the client.

<strong>Email:</strong>
Enter the email address of the client.

<strong>Fax:</strong>
Enter the fax number of the client.

<strong>Po Box No:</strong>
Enter the post box number of the client.
